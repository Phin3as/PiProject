# PiProject

##Step 1: IDE
Install an IDE. I use Intellij IDEA. Also, install scala(2.11) .

##Step 2: Git
Install Git

##Step 3: Maven
Install maven

##Step 4: How to run
* Ask me for repository access or fork the project
* do - git clone <repo name> a.k.a git clone https://github.com/Phin3as/PiProject.git
* mvn clean package
